Summary

My Novel Garden is a gardening app that monitors the light and moisture levels for your garden.

The following technologies will be used:

  • Arduino
  • Sparkfun
  • Angular
  • Express

APIs

  1. The app will use an API that feeds data back from the arduino sensors.
  2. The app will use an API with growning info for data to data comparison

Sensors

The app will monitor the following levels:

  • Moisture
  • Light
  • Temperature
  • Fertilization levels (stretch)

Stretch Goals

The app will be connected to a watersource and LED to supplement plant levels.

Eventually adjustments will be automated based on the specific type of plant.
